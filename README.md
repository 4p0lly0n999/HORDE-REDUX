# Horde: REDUX Gamemode

## What is Horde: REDUX ?
Horde: Redux (Horde:R) is a modification for Gmod gamemode Horde. 
The idea is to completely rewrite the gamemode's code and make it as modular and moddable as possible, 
thereby improving the gaming and modding experience for both players and creators!
